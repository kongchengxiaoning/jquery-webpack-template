# webpack5-template

### PROD
* [x] [jquery](https://jquery.com)

### DEV
* [x] [webpack5](https://webpack.docschina.org)

## Usage
### install dep
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### preview the release environment effect
```
yarn serve
```
